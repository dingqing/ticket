# ticket
出票小应用，适用于演唱会或其他活动出票。


## 系统介绍
### 功能
- 初始化座位票
- 查询余票
- 购票
- 查询“未完成”/“已完成”订单
- 取消订单
- 系统取消超时订单
### 用户体验
- 尽量让用户买到座位相邻的票
- 座位不相邻原因：
    - 自然不相邻：跨区/跨行
    - 夹在中间的票已经出售
- “随机出票”会影响座位相邻
    - 如果要求随机出票，那么就会导致下一次出票有可能不相邻，除非每次都从剩余票的第一张开始出票
